# Sobre
Palpiteiro utiliza um algoritmo preditivo de machine learning e um algoritmo genético, 
alimentados por dados oficias do Cartola FC e sites de apostas esportivas, para 
encontrar a melhor escalação possível para o seu time no Cartola FC.

Palpiteiro é um projeto open-source escrito inteiramente em 
Python e está disponível em:
[github\matheusccouto\palpiteiro](https://github.com/matheusccouto/palpiteiro)
